# BIOF050-Project
Contains all relevant code and information for the FAES BIOF050 Spring 2026 workshop

Project architecture based on the network developed by Eraslan, et. al:

Eraslan, G., Simon, L.M., Mircea, M. et al. Single-cell RNA-seq denoising using a deep count autoencoder. Nat Commun 10, 390 (2019). https://doi.org/10.1038/s41467-018-07931-2
